A friend of mine who teaches ESL told me that learning slang in a conversational context is something that is currently lacking in language curricula. thenkyou is a full-stack Sinatra app designed to address this problem.

\\

Users are provided with readings of different skill levels. Highlighted focus terms will give definitions and usage examples when interacted with. These readings and vocab terms are linked through a relational database.

//

An optional signup feature gives additional readings. It also tailors displayed content to the user's language, so that they are not given readings to study in their native language.

\\

thenkyou currently includes readings in English, German, and Japanese, but it would be cool to add more content in different languages soon.

Rep Request is a pair-programmed Sinatra app that provides information about Congress members. It uses the govtrack.us API and an open-source database to gather information.

//

Congress members are organized by state. We utilized the Google Civic Information API to let users find representatives based on their home address, as well.

\\

Various information is displayed upon clicking a politician. LESS and Flexbox were used to help organize and style the information.

badnname is a full-stack, pair-programmed Express app. It tells users if their idea for a new band name has already been taken.

Check out an early build here.

\\

It grabs information via the Discogs API and a Bandcamp scraper. If the bandname is taken, information and links to the existing band are displayed to the user.

//

User searches are saved to a database (we used MongoDB) for future reference.
